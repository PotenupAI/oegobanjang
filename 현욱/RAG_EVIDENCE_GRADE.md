# RAG Evidence Grade 분류 기준

## 개요

외고반장 프로젝트의 RAG 시스템에서 수집한 문서들을 신뢰도 기준으로 분류합니다.

**핵심**: Grade는 난이도가 아니라 **증거 근거의 법적 신뢰도**를 나타냅니다.

---

## Grade 분류표

| Grade | 명칭 | 법적 효력 | 출처 | 역할 |
|-------|------|---------|------|------|
| **A** | 공식 법령 | 최상 | 법제처, 국회 | 법적 구속력 있음 (법원·행정기관 인정) |
| **B** | 공식 절차/안내 | 중상 | 정부 기관 (EPS, 정부24, 출입국청) | 정부 공식 가이드 (강제성 없음) |
| **C** | 서식/양식 | 중 | 법제처, 정부부처 | 공식 템플릿 (참고용) |
| **D** | 안전·참고 자료 | 중하 | 산업인력공단, 교육 기관 | 교육/보조 자료 (정보 제공용) |

---

## 각 Grade별 상세 설명

### Grade A: 공식 법령 (법적 근거)

**특징**:
- 국회에서 입법하고 법제처에서 공식 발행
- 법원과 행정기관이 인정하는 최고 수준의 근거
- 법적 구속력 있음 → AI가 "반드시" 따라야 함

**예시**:
- 출입국관리법
- 출입국관리법 시행령
- 출입국관리법 시행규칙
- 외국인근로자고용법
- 외국인근로자고용법 시행령
- 외국인근로자고용법 시행규칙

**Task**: Task 1 (`law_crawler.py`)

**사용 예**:
```
"E-9 비자 체류연장에 필수 서류가 뭐예요?"
→ [출입국관리법 시행규칙 제72조] 체류기간 연장 신청 시 제출 서류:
   - 여권사본
   - 외국인등록증 사본
   - 고용허가서 사본
   - ...
```

---

### Grade B: 공식 절차/안내 (정부 기관 가이드)

**특징**:
- 정부 기관(EPS, 정부24, 출입국청)이 공식으로 발행
- "법령 해석"이 아니라 "운영 절차" 설명
- 강제성은 없지만 정부 권장 방식임
- 실무자가 따르는 표준 절차

**예시**:
- EPS 포털의 "E-9 체류연장 절차"
- EPS 포털의 "E-9 사업장변경 절차"
- 정부24의 "체류기간 연장 안내"
- 출입국청의 "H-2 갱신 절차"

**Task**: Task 2 (`eps_crawler.py`), Task 3 (`gov24_crawler.py`)

**사용 예**:
```
"E-9 체류연장을 신청하려면 어떻게 해야 해요?"
→ [EPS 포털 - E-9 체류연장 절차]
   1단계: 사업주가 EPS 홈페이지에서 사전 신청
   2단계: 외국인이 필요 서류 준비
   3단계: 출입국청에 신청
   ...
```

---

### Grade C: 서식/양식 (공식 템플릿)

**특징**:
- 법제처, 정부부처에서 공식 양식 제공
- "어떤 내용을 써야 하나" 를 알려줌
- 법적 효력은 없지만 표준 형식임

**예시**:
- 표준근로계약서 (국문 + 다국어 5종)
- 체류연장 신청서 (별지 34호)
- 고용허가서 양식

**Task**: Task 3 (`gov24_crawler.py`)

**사용 예**:
```
"표준근로계약서에는 뭐가 들어가야 해요?"
→ [표준근로계약서 양식]
   - 계약 기간
   - 월급여
   - 근무시간
   - 휴가/휴직
   ...
```

---

### Grade D: 안전·참고 자료 (교육/보조)

**특징**:
- 산업인력공단, 교육기관 등에서 발행
- 안전교육, 생활 정보, 참고용
- 법적 근거는 아니지만 알아두면 좋은 정보

**예시**:
- 외국인 기초안전보건교육 자료
- 다국어 안전표지
- 외국인 근로자 생활 안내
- 상담센터 안내

**Task**: Task 4 (`hrd_crawler.py`)

**사용 예**:
```
"외국인 근로자가 알아두면 좋은 안전 상식이 뭐예요?"
→ [산업인력공단 - 기초안전보건교육]
   - 작업장 위험 요소 인식
   - 개인보호장비 착용법
   ...
```

## RAG 검색 우선순위

비자 에이전트가 사용자 질문에 답할 때 **Grade 순서대로** 근거를 찾습니다.

### 검색 흐름

```
사용자 질문
  ↓
Grade A (법령)에서 검색
  ├─ 찾음 → 답변: "[법령명] 제N조에 따르면..."
  ├─ 못 찾음 → 다음 단계
  ↓
Grade B (공식 절차)에서 검색
  ├─ 찾음 → 답변: "[정부기관] 절차에 따르면..."
  ├─ 못 찾음 → 다음 단계
  ↓
Grade C (서식)에서 검색
  ├─ 찾음 → 답변: "[공식 양식]에 따르면..."
  ├─ 못 찾음 → 다음 단계
  ↓
Grade D (참고 자료)에서 검색
  ├─ 찾음 → 답변: "[교육 자료]에 따르면..."
  ├─ 못 찾음 → 최종 단계
  ↓
모든 Grade에서 못 찾음
  → 답변: "공식 근거를 찾지 못했습니다. 행정사 또는 노무사 검토가 필요합니다."
```

## 주의사항

### Grade ≠ 난이도

- Grade A가 "어렵다"는 뜻이 아님
- Grade는 "공식성"과 "법적 효력"을 나타냄
- Grade D가 "쉽다"는 뜻도 아님

### Grade가 낮다고 무시하지 않기

- Grade D 자료도 근로자 안전에 중요함
- 에이전트는 모든 Grade 자료를 종합해서 판단해야 함
- 단, **법적 판단이 필요한 경우는 Grade A부터 참고**

### 검색 실패 시 정직하기

- "공식 근거를 찾지 못했습니다"라고 명확히 말하기
- 근거 없이 추측 답변 금지
- 행정사/노무사 검토 권고
