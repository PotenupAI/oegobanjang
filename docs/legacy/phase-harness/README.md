# 레거시 Phase 하네스 아카이브

이 폴더는 새 `missions/` 구조로 전환되기 전의 phase 기반 하네스 문서를 보관한다.

보관 목적:

- 예전 phase 의사결정 과정을 추적하기 위해서
- 새 구조의 docs/mission 문서와 1:1 대응을 확인하기 위해서
- 이전 phase 구현 의도를 잃지 않기 위해서

구성:

- `docs/legacy/phase-harness/ENGINEERING_PLAN.md`
- `docs/legacy/phase-harness/PHASE3_REVIEW_NOTES.md`
- `docs/legacy/phase-harness/REMAINING_EXECUTION.md`
- `docs/legacy/phase-harness/phases/...`

이 아카이브는 실행 대상이 아니다. 현재 구현은 새 구조의 `missions/`, `backend/`, `data-pipeline/`, `evals/`, `docs/`를 기준으로 진행한다.
